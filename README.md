
# Portfolio Website

A premium, dark-themed portfolio built with modern web technologies.

## Features

-   **Interactive UI**: 3D elements, glassmorphism, and smooth animations using Framer Motion.
-   **Contact Section**: Integrated 3D illustration and functional form with EmailJS.
-   **Timeline**: Custom "Full-Drop" animation for the experience section.
-   **Responsive**: Fully optimized for desktop and mobile devices.

## Tech Stack

-   **Frontend**: React, Vite
-   **Styling**: Tailwind CSS
-   **Animations**: Framer Motion
-   **Icons**: Lucide React
-   **Email**: EmailJS

## Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/Kartikgupta-Github/portfolio.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
